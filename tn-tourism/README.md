# 🌍 Tamil Nadu Tourist Places Website

A simple responsive web project showcasing famous tourist places in Tamil Nadu using HTML and CSS.

---

## 📌 Project Description  

- This project displays popular tourist destinations in Tamil Nadu with beautiful background images, hover effects, and a grid layout design.

- It is built as a practice project to improve frontend development skills.

---

## 🚀 Features

- Responsive grid layout  
- Full-screen background images for each place  
- Hover overlay effect  
- Clean UI design  
- Simple navigation-style marquee header  

---

## 🏞️ Places Included  

- Thanjavur   
- Courtallam  
- Dhanushkodi  
- Madurai  
- Kodaikanal  
- Mahabalipuram  
- Coimbatore  
- Kanyakumari  
- Ooty  
- Chidambaram  
- Tuticorin  
- Kolli Hills  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- CSS Grid  
- Flexbox
