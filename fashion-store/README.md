# 👗 Fashion Store - HTML & CSS Project  

## 📌 Project Overview

Fashion Store is a simple frontend web page built using HTML and CSS.  
It is designed as a basic e-commerce UI with product categories, promotional offers, and informational sections.  

## 🚀 Features   

🛍️ Product category section (T-shirt, Shirt, Shoes, etc.)  
🎨 Clean navigation bar  
🖼️ Banner image with animation text  
📊 Festival sales table with discounts    
📖 Informative content section  
📞 Footer section  

## 🛠️ Technologies Used  
- HTML5  
- CSS3
